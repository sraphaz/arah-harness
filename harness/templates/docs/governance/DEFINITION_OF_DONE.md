# Definition of Done

Critérios mínimos antes de merge (profile consulting).

## Todo PR

- [ ] Revisão humana registrada
- [ ] Checks do CI verdes
- [ ] Sem segredos no diff
- [ ] Mudança descrita no PR (o quê e por quê)
- [ ] Escopo rastreável a spec ou ADR quando aplicável

## Profile consulting

- [ ] Spec correspondente atualizada e válida (`docs/specs/`)
- [ ] Escopo rastreável ao proposal-scope aprovado (quando handoff)
- [ ] ADR para decisão estrutural nova
- [ ] Domain agents consultivos acionados em PRs de arquitetura/testes quando relevante

## Handoff

- [ ] Gate `handoff_acceptance` aprovado quando entrega para cliente ou repo downstream
