# Harness profiles

Profiles declarativos para `install-harness.ps1`. Genéricos — sem marca de consultoria.

| ID | Uso |
|----|-----|
| `minimal` | Base: PR, manifests, CI mínima |
| `consulting` | Spec-before-work, gates de handoff |
| `product` | Produto com releases |
| `enterprise` | Compliance, stakeholders |
| `open-source` | Contribuição pública |

Scripts: `harness/scripts/` · Schemas: `docs/schemas/`
